## Textractor
easy to use CLI tool for extracting text from images,

using tesserract.

### Dependencies 
- pytesseract module

## Installation for Unix & Linux
- go to /tmp `cd /tmp`
- clone the repo `git clone https://github.com/AmitNiz/toolBox.git`
- go to toolbox/textractor `cd toolbox/textractor`
- run `./install.sh`
